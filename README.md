# EXPROSA 2026 Landing Page

Landing page baru untuk PT Exprosa Global Nusantara dengan arah visual premium tech, dark-clean, motion-first, dan siap deploy ke Vercel.

## Stack

- Next.js App Router
- Tailwind CSS
- Motion
- GSAP ScrollTrigger
- Vercel

## Jalankan lokal

```bash
npm install
npm run dev
```

Buka `http://localhost:3000`.

## Build production

```bash
npm run build
npm run start
```
